# Spring Boot and Role Based Spring Security tutorial

**springboot-security-demo**: This POC demonstrates how to secure a SpringBoot based web application using :
- Spring Boot
- Spring MVC
- Role Based Spring Security
- JPA
- Thymeleaf
- MySQL

#### How to run?

springboot-security-demo> mvn spring-boot:run

Go to http://localhost:8080/
